Documentation

I began by creating a high-frequency noise texture and used Ramp and Level TOPs to increase the contrast, turning it into a sharper mask-like texture. Then, I introduced a second Noise TOP with a lower frequency to generate larger random forms. Using Displace TOP, I used the first noise texture to distort the second one, creating more irregular and edges. Finally, I refined the composition by adjusting the scale and positioning with Transform TOP before sending it to the final output. To create movement, I replaced static values in the Translate parameters using absTime.frame. The ty value creates a fast vertical movement. This makes the texture feel like it is flowing downward. I also adjusted the tz value to create smoother transformations within the 3D noise space, this allows the shapes to evolve.

Documentation Block II

In this iteration, I used a Feedback Loop and a Tile TOP. This transforms the visuals from isolated shapes into a more complex and repetitive pattern. This created a rhythmic digital texture that feels more layered and dynamic. To create movement, I added (absTime.seconds * 10) to the Rotate parameter, this makes the texture to continuously spin over time. I also applied non-uniform scaling (1.5, 1.1) to stretch the feedback trails horizontally.

Documentation

I created this dreamy, planet-like texture by first using a Ramp TOP to shift the original noise into softer pink, purple, and off-white tones, giving it a biological feeling. I then used a Limit TOP to simplify the gradients into flatter areas of color. Through a Feedback loop, I added slight scaling and rotation in the Transform TOP, which allowed the texture to continuously grow and twist over time. Finally, I used a subtle Displace effect to add more surface details, creating a texture that feels similar to a planet or material.

Study 10Motion
Documentation

In this exploration, I introduced audio-reactive controls to connect sound with the visuals. I used an Audio File In CHOP to bring in sound, then passed it through Analyze, Filter, and Math nodes to extract and smooth the audio data. This allowed the texture to respond to sound in real time. I also added a sine wave expression (math.sin(absTime.seconds)) to the Z-axis of the Noise TOP, which creates a breathing motion. Finally, I used Emboss and Threshold TOPs to add depth and create a rough surface texture.

For this texture, I used an Edge TOP to highlight the brightness change, this creates outlines that resemble cracks texture. I layered multiple noise textures together using an Add TOP. After applying Blur and Level nodes, I increased the contrast to create a black-and-white, ink-like texture, then used a Ramp TOP to change to a blue tones. To animate the piece, I adjusted the noise seed and movement speed so the smaller particles flicker quickly while the larger shapes move more slowly. This created the motion look like a feeling of seaside reefs.

In this iteration, I added a Rectangle TOP and rotated it by 45 degrees to create a diamond shape that acts as a visual frame. I adjusted the Fill Alpha and Background Alpha so the texture could partially show through the shape (layered effect). I also added an expression to the Border Width using math.sin(absTime.seconds * 3) to make the border pulse over time. Before combining the geometry with the texture, I used a Threshold TOP to create a strong black-and-white background, which helped the green diamond stand out more clearly.

I started by generating a noise texture. Using Displace TOP, I used the grain texture to distort the larger forms, creating more material-like surfaces. I started with a vertical gradient using brown and green tones to create a metallic background. I then used a Mirror TOP to duplicate and reflect the gradient, creating a more balanced composition. Also, I added a Blur TOP softened the transitions between colors, making the texture feel more atmospheric. To create contrast, I added a semi-transparent green circle on top as a floating focal point. I animated the circle by using a Python expression (0.5 + math.sin(absTime.seconds * 0.5)) in the Center X parameter, this allows it to move smoothly back and forth across the screen..

Documentation: Physical Space

Testing the digital materiality on physical geometry using Kantan Mapper... I wanted to project three moving visuals onto walls at different distances, so I used Kantan Mapper to divide and map each animation into specific projection areas. Since the walls were positioned at different depths, I adjusted each mapped shape to fit the physical surfaces. This process allowed me to move the visuals beyond the screen and explore how digital textures interact with architectural space, scale, and viewer perception in a physical environment.
