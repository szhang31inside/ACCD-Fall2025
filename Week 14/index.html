Process

1. My first step was to connect four buttons to my board and make sure they worked properly. During this process, I encountered several problems. When I opened the Arduino IDE, only two values (0, 0) appeared, which meant that the third and fourth buttons were not being detected. I tried fixing this by changing both the code and the wiring. For example, I checked that the condition was set to values.length == 4 instead of values.length == 2. I attempted multiple code changes, but eventually I realized the issue was that I didn't upload the code, I only checked.
